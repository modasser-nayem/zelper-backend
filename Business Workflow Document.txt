Project Overview
Goal
Someone Help is a location-based platform that connects people who need immediate or scheduled assistance with nearby people willing to help.
Unlike traditional marketplaces, users do not have permanent customer or provider roles. Every account is a standard user account that can request help or provide help at any time.
The platform prioritizes:
Privacy
Trust
Simple negotiation
Secure escrow payment
Easy onboarding

Core Principles
1. Single User Account
Every registered account is simply a User.
A user can
Request help
Provide help
Do both simultaneously
No separate registration is required.

2. Optional Verification
Verification is optional.
Users may request identity verification whenever they choose.
Verification increases trust but never blocks platform usage.
Verified users receive a badge that is visible during provider selection.

3. Privacy First
Before payment, both users remain anonymous.
The platform hides
Name
Photo
Phone Number
Email
Exact Address
Exact GPS Location
Customers evaluate helpers only using trust-related information.
